This is a simple web application where users can add and view posts with a photo, title, description, and location.

Features

Add new posts with:
An image
A title
A description
A location
View all posts
Data is stored in MongoDB
Built using Node.js, Express, MongoDB, and Pug as the view engine

Project Structure

Technologies Used

Node.js
Express.js
MongoDb


Each post includes:
Photo
Title
Description
Location
